# VouchX Wiki Source

This folder contains the help pages for publishing as the VouchX usage wiki.

Recommended reading order:

1. `home.md`
   Purpose: landing page for moderators and members
   Suggested wiki title: `VouchX Help`
2. `moderator-quick-start.md`
   Purpose: first-time setup and launch checklist
   Suggested wiki title: `Moderator Quick Start`
3. `member-guide.md`
   Purpose: member-facing submission and status guide
   Suggested wiki title: `Member Guide`
4. `moderator-guide.md`
   Purpose: day-to-day moderation workflow
   Suggested wiki title: `Moderator Guide`
5. `settings-reference.md`
   Purpose: install settings, templates, themes, and placeholders
   Suggested wiki title: `Settings Reference`
6. `troubleshooting.md`
   Purpose: common issues, retention, and operational FAQ
   Suggested wiki title: `Troubleshooting`

Why this structure:

- `home.md` is short and only routes readers to the right page.
- `moderator-quick-start.md` covers first launch without forcing readers through every reference detail.
- `member-guide.md` and `moderator-guide.md` separate the two main audiences.
- `settings-reference.md` keeps configuration details in one place so the workflow pages can stay task-focused.
- `troubleshooting.md` collects edge cases, limits, and retention notes that would otherwise get repeated.

Keep legal pages separate from this help set:

- [`TERMS_OF_SERVICE.md`](../../TERMS_OF_SERVICE.md)
- [`PRIVACY_POLICY.md`](../../PRIVACY_POLICY.md)
